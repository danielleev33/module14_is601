This project gave me a good understanding of how parts of a web application connect in practice. I was able to work with authentication, database operations, full BREAD functionality, testing, Docker, and CI/CD in one project.

The most useful part for me was seeing how small issues CAN break the whole app even when the code looked fine to me. I had to troubleshoot problems with dependencies, JWT authentication, Docker builds, Swagger authorization, Redis checks, and database setup. Working through those issues helped me understand the importance of testing and debugging step by step instead of making random changes.

Building the additional profile feature also helped me see how to extend an existing application without overcomplicating it. I had to think about route design, validation, authentication, and how to make the feature fit with the rest of the app.

This project made me more comfortable with the full development workflow; from coding and testing to deployment. It also showed me how important it is to be patient when debugging bc many of the hardest issues were caused by setup and integration details, not just the application logic itself.
